# 仓库管理系统

  一款分布式仓库管理系统，采用分布式 [zheng](https://github.com/shuzheng/zheng) 框架执行架构，目前具备 “权限管理模块系统” 和 ”库存管理模块系统“，功能分离，业务解耦。
